YES Command
---
Unix has `yes` command that print a line of spesified string or a `y` repeatedly until killed. 
This command is written by David MacKenzie. 

example:
`yes` will output:

```
y
y
y
y
y
y
.... (still continue)
```

`yes man` will output:

```
man
man
man
man
man
```
